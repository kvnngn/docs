## Welcome to BETiq

This guide will help you get started with BETiq quickly. Follow these steps to set up your environment and run your first data analysis.

## Prerequisites

Before you begin, ensure you have:

<CardGroup cols={2}>
  <Card title="Access Credentials" icon="key">
    Valid L'Oréal credentials and BETiq access permissions
  </Card>
  <Card title="Network Access" icon="network-wired">
    Connection to L'Oréal's internal network or VPN
  </Card>
</CardGroup>

## Step 1: Environment Setup

<AccordionGroup>
  <Accordion icon="user-check" title="Authentication">
    1. Contact your system administrator to obtain BETiq credentials
    2. Ensure your account has the appropriate permissions
    3. Test your credentials by accessing the BETiq platform
  </Accordion>
  <Accordion icon="globe" title="Access BETiq Platform">
    1. Navigate to the BETiq platform URL (contact your admin for the specific URL)
    2. Log in using your L'Oréal credentials
    3. Verify you can see the main dashboard
    
    <Tip>Bookmark the platform URL for easy access</Tip>
  </Accordion>
</AccordionGroup>

## Step 2: Your First Data Query

<AccordionGroup>
  <Accordion icon="database" title="Connect to Data Sources">
    1. Navigate to the **Data Sources** section
    2. Select from available pre-configured data sources
    3. Test the connection to ensure data access
  </Accordion>
  <Accordion icon="chart-line" title="Create Your First Dashboard">
    1. Go to **Dashboards** → **New Dashboard**
    2. Select a data source from Step 1
    3. Choose visualization type (chart, table, etc.)
    4. Configure your metrics and dimensions
    5. Save your dashboard
    
    <Tip>Start with simple visualizations and build complexity gradually</Tip>
  </Accordion>
</AccordionGroup>

## Step 3: Explore Advanced Features

<Accordion icon="rocket" title="Next Steps">
  1. Explore the **Analytics** section for advanced analysis tools
  2. Set up automated reports for regular insights
  3. Configure alerts for key metrics
  4. Share your dashboards with team members
</Accordion>
